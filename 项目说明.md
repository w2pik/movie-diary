# 🎬 电影日记 — 项目说明

## 这是什么

个人观影记录工具。226 部电影数据来自豆瓣导出的 xlsx，自动从 Wikipedia/TMDb 爬取海报封面和元数据。Web 界面浏览、筛选、统计、添加新电影。

## 启动方式

### 一键启动（推荐）

双击 `start.bat` → 自动启动服务 + 打开浏览器

### 命令行启动

```bash
PYTHONIOENCODING=utf-8 python app.py
# 浏览器打开 http://127.0.0.1:5000
```

### Python 路径

```
D:/03_Software/UV_Tool/python_standalone/cpython-3.14.5-windows-x86_64-none/python.exe
```
由 UV 包管理器管理，安装包需加 `--break-system-packages`。

## 功能清单

### 🎞️ 电影墙
- 网格展示所有电影海报卡片
- 搜索：按电影名/导演/演员搜索
- 筛选：年份、地区、类型、最低评分
- 排序：观看时间、评分、年份、片名
- 分页加载

### 📊 统计分析（顶部标签切换）
- 年度观影柱状图 + 累计趋势线
- 月度观影（可切换年份）
- 累计观影趋势曲线
- 星期分布柱状图
- 观影热力图（月份 × 星期）
- 类型分布 Top 15（横向柱状图）
- 国家分布（环形饼图）

### 📋 电影详情（点击卡片）
- 完整信息：导演/主演/简介/评分/片长/语言/国家/上映日期...
- 空字段显示"暂无"
- 标签点击筛选
- **🗑️ 删除此电影** 按钮
- **🪄 补全信息** 按钮（自动从 Wikipedia/TMDb 填充缺失字段）

### ➕ 添加电影（右上角按钮）
- 填写电影名即可提交
- 自动从 Wikipedia + TMDb 获取：封面、年份、评分、导演、主演、类型、简介、国家、语言...
- `Ctrl+Enter` 快捷提交

### ⌨️ 键盘快捷键
- `/` — 聚焦搜索框
- `Esc` — 关闭弹窗
- `Ctrl+Enter` — 提交新增表单

## 封面和元数据抓取策略

```
1. 中文 Wikipedia (优先) → 导演/主演/编剧/国家/语言/简介(中文)/类型(中文)
2. 英文 Wikipedia (补位) → 原片名/上映日期/片长
3. TMDb 直接搜索 (补位) → 评分/封面图
```

- 首次导入时批量爬取（2 秒间隔）
- 手动添加电影时后台自动抓取
- "补全信息"按钮随时可重新抓取

### 常见抓取失败原因
- Wikipedia 没有该电影的页面 → 手动填写
- TMDb 搜索没结果 → 用英文原名重新搜索
- 封面下载失败 → 网络问题，重试即可

## API 路由

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/movies` | 电影列表（支持 q/year/country/genre/tags/min_rating/sort_by/page） |
| GET | `/api/movie/<id>` | 单部电影详情 |
| POST | `/api/movies/add` | 新增电影（自动抓封面+元数据） |
| POST | `/api/movie/<id>/enrich` | 补全电影信息 |
| DELETE | `/api/movie/<id>` | 删除电影 |
| GET | `/api/stats` | 基础统计 |
| GET | `/api/stats/time` | 时间粒度统计 (granularity=year/month/week/day) |
| GET | `/api/stats/heatmap` | 观影热力图数据 |
| GET | `/api/filters` | 筛选选项（标签/年份/国家/类型） |
